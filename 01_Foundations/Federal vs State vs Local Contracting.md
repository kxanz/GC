Federal Contracting:

**What is it?**

Contracts with U.S Federal Agencies.

- Largest budgets 
- Long-term, stable revenue
- set-aside 

**Location**

[Sam.gov](https://sam.gov)

---

State Contracting:

**Contract Size**

- 10k 

**What is it?**

Contracts with State Governments.

- Each state has its own rules 
- Best for Saas, web apps, IT consulting, data systems

**Location**

State Procurement portals (e.g, Cal eProcure, Texas SmartBuy)

---

Local Contracting:

**What is it?**

Contracts with cities, counties, school districts, and utilities.
